## 📖 About These Notes

This repository is a **complete, interview-focused study kit** for anyone preparing for a
**Java Backend Developer** role — freshers and experienced developers alike.

Everything is explained **as if teaching a beginner**: simple words, small topics, real examples,
diagrams, and a **quick revision summary** at the end of every topic. Perfect for **learning** and
for **fast revision** the night before an interview.

**What makes it different:**
- ✅ Very simple English — no heavy jargon.
- ✅ Every topic is short and easy to revise.
- ✅ ASCII/Mermaid diagrams to *see* the concept.
- ✅ Real-world use cases + common interview questions.
- ✅ Common mistakes, best practices, and important keywords.
- ✅ Difficulty level + interview frequency on every topic.
- ✅ 500+ interview questions with answers.
- ✅ 8 full backend project architectures.
- ✅ Company-wise question patterns for 20 companies.

## 🖨️ Make a PDF (Offline Reading)

Turn all notes into one PDF. Full guide in `CheatSheets/HOW-TO-MAKE-PDF.md`.

**Quickest way (with [Pandoc](https://pandoc.org/) installed):**

```powershell
# Run inside the Backend-Interview-Notes folder
$files = Get-ChildItem -Recurse -Filter *.md | Sort-Object FullName
pandoc $files.FullName -o Backend-Interview-Notes.pdf --toc --toc-depth=2 -V geometry:margin=1in
```

**Easiest way (VS Code):** install the **"Markdown PDF"** extension → right-click a `.md` file →
*Export (pdf)*.
