## 🎯 Golden Rules for Java Interviews

- Always give a **1-line definition first**, then explain with an example.
- Mention **real-world use** to show you actually understand.
- Know the difference between **similar things** (e.g., `==` vs `equals`, ArrayList vs LinkedList).
- Be ready to **write small code** on paper / whiteboard.
