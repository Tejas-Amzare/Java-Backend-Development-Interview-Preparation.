## 🎯 How to Use
1. Cover the answer, read the question, answer aloud.
2. Mark ones you miss; revise those the next day.
3. Focus on 🔥 Very High frequency topics first.
4. Practice the **🎤 Detailed Spoken Answers** out loud — record yourself and improve delivery.

## 🗣️ How to Speak in an Interview (Golden Rules)
- **Structure every answer:** Definition → Example → Why it matters.
- **Don't rush** — a calm 45-second answer beats a rushed 15-second one.
- **Use real backend examples** (users, orders, payments) — shows practical understanding.
- **For "difference between X and Y"** — always give a clear verbal comparison.
- **If unsure:** *"I haven't used that directly, but my understanding is…"* — never bluff or go silent.
