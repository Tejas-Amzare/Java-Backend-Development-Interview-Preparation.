# 🏢 Company-Wise Interview Questions

Frequently asked backend interview questions grouped by company, based on common patterns.

## 📄 Guides
- [Product Companies (FAANG + Product)](./01-Product-Companies.md) — Amazon, Google, Microsoft, Oracle, Walmart, Flipkart, Swiggy, Zomato, Razorpay, PhonePe
- [Finance & Service Companies](./02-Finance-and-Service.md) — JPMorgan, Goldman Sachs, Deloitte, Accenture, Infosys, TCS, Capgemini, Cognizant, IBM, Zoho

## 🎯 General Advice
- **Product companies** (Amazon, Google) → heavy DSA + system design.
- **Service companies** (TCS, Infosys) → core Java, SQL, projects, HR.
- **Finance** (JPMorgan, Goldman) → DSA + Java internals + system design + low latency.
