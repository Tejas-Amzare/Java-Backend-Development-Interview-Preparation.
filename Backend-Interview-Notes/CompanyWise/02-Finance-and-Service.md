# Finance & Service Companies — Interview Questions

---

## 💰 Finance Companies (High Bar)

### 🏦 JPMorgan Chase
**Focus:** DSA + Core Java + System Design + some finance basics.
- DSA: arrays, strings, trees, DP (medium).
- Java: multithreading, collections, OOP, exception handling.
- System design: trading/transaction systems, scalability.
- SQL: joins, transactions, ACID.
- Project deep-dive + behavioral.

### 🏛️ Goldman Sachs
**Focus:** Strong DSA, Java internals, low-latency, math/logic.
- DSA: medium-hard (graphs, DP, sorting).
- Java: JVM, GC, concurrency, memory model.
- System design + object modeling (LLD).
- Aptitude + probability sometimes.
> 💡 Know **Java internals** deeply (how HashMap/GC/threads work).

---

## 🏢 Service Companies (Core-Focused)

> Service companies (TCS, Infosys, etc.) focus more on **core Java, SQL, projects, and HR**
> than hard DSA. Fundamentals + communication matter most.

### 🔷 TCS (incl. TCS NQT / Digital)
- Core Java: OOP, collections, exceptions, strings.
- SQL: queries, joins, normalization.
- Aptitude + coding (easy-medium).
- HR: why TCS, relocation, bond.

### 🔷 Infosys
- Java basics, OOP, collections.
- SQL + DBMS concepts.
- Coding (easy-medium: arrays, strings, patterns).
- HR + communication.

### 🔷 Capgemini
- Java, SQL, OOP.
- Pseudocode round + aptitude.
- Basic coding + HR.

### 🔷 Cognizant
- Core Java, collections, exceptions.
- SQL joins, subqueries.
- Spring/Spring Boot basics for experienced roles.
- HR + project discussion.

### 🔷 Accenture
- Java fundamentals, OOP.
- SQL, DBMS.
- Aptitude + communication + coding (easy).
- HR: adaptability, teamwork.

### 🔷 IBM
- Java, OOP, collections, exceptions.
- SQL + data structures.
- Cloud/DevOps basics (for some roles).
- Cognitive aptitude test.

### 🟢 Zoho
**Focus:** Strong coding + logic (harder than typical service companies).
- Multiple coding rounds (arrays, strings, patterns, logic).
- Data structures + real problem-solving.
- OOP design.
- Technical + HR.
> 💡 Zoho is coding-heavy — practice implementation problems thoroughly.

---

## 📋 What to Prepare by Company Type
| Type | Priority |
|------|----------|
| Finance (JPMC, GS) | DSA + Java internals + system design |
| Service (TCS, Infosys, Accenture) | Core Java + SQL + projects + HR + aptitude |
| Zoho | Coding + logic (heavy) |

## 🎯 Universal Prep Checklist
- ✅ Core Java (OOP, collections, exceptions, multithreading)
- ✅ SQL (joins, normalization, queries)
- ✅ DSA (arrays, strings, LinkedList, trees — at least easy-medium)
- ✅ Spring Boot basics (for backend roles)
- ✅ Your projects (be able to explain deeply)
- ✅ HR answers (STAR + company research)
