## 🎯 How to Answer a System Design Question
1. **Clarify requirements** (functional + non-functional).
2. **Estimate scale** (users, requests/sec, storage).
3. **Draw high-level design** (client → LB → services → DB/cache).
4. **Deep dive** into key components.
5. **Discuss trade-offs** (consistency vs availability, cost, latency).
